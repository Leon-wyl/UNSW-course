# Software Engineering Fundamentals
- Course Code: COMP1531
- Prerequisite: COMP1511
- Lecture: Hayden Smith
- Contact Hour: 7 (2*2 hrs live session + 1 hr tutorial + 2 hrs lab)
- Language used: Python
- Assessment: Project(50%) + Class Mark (Tutorials + Labs) (20%) + Final (30%)
- Difficulty: 2.5/5 for the content, 3.5/5 for the Final, Project difficulty varies due to the capability of your teammates and you
- Trimester when I did it: 20T3

This course provided an introduction of software engineering principles like Agile development model, through a project aiming to build a backend server for a web app of team collaborating app. It is recommended to find good teammates prior to the course or you would probably have incapable teammates and even freerider. Students also learned more advanced Python programming skills in this course. The final of this course is very hard but it has fairly tight time limit.
